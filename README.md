##  Run this App

* Clone this project

```
git clone https://github.com/arcanine525/MusicAppDemo.git
```

* Don't forget to install `node_modules` by using:

```
npm install
```

* Android

```
react-native run-android
```
* iOS

```
react-native run-ios
```

* NOTE: We haven't tested on iOS yet
